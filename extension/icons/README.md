# Icons

Place your extension icons here:

- `icon16.png` - 16x16 pixels (toolbar)
- `icon48.png` - 48x48 pixels (extension management page)
- `icon128.png` - 128x128 pixels (Chrome Web Store)

You can create these icons using any image editor or online tool.

Recommended design: Cookie icon with a shield or lock symbol to represent security.

For now, you can use placeholder icons or generate them using tools like:
- https://www.favicon-generator.org/
- https://realfavicongenerator.net/
- Figma, Sketch, or Adobe Illustrator

To test without icons, Chrome will use a default placeholder.
